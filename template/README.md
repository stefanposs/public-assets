# [Topic Name] – Public Assets

All publicly shareable media assets for the [Topic Name]:

- `logos/`          → Logos for the company, brand, or initiative  
- `photos/`         → Profile pictures, team photos, portraits  
- `screenshots/`    → Product images, website screenshots  
- `social/`         → Social media images, banners  
- `press/`          → Press photos, media kits  

## Usage

- Assets can be publicly shared.  
- Only use approved versions of third-party logos or images.  
- Do not add sensitive or internal files.

**How to use this template:**

1. Copy the entire `template/` folder
2. Rename it to your topic name (e.g., `my-company`, `my-project`)
3. Update this README.md with your topic name
4. Add your assets to the corresponding subfolders
